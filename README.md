# FRever — Dein intelligenter Lebensmittel-Manager

**FRever** ist eine innovative Lösung zur Reduzierung von Lebensmittelverschwendung im Haushalt. Die Anwendung hilft Nutzern, ihre Vorräte effizient zu verwalten und Geld zu sparen.

## Hauptfunktionen
- **Kassenbon-Scanner:** Schnelles Hinzufügen von Produkten durch automatische Texterkennung.
- **Haltbarkeits-Tracker:** Intelligente Benachrichtigungen 2 Tage vor Ablauf des Mindesthaltbarkeitsdatums.
- **Inventar-Analyse:** Grafische Auswertung der verbrauchten und weggeworfenen Lebensmittel.
- **Nachhaltigkeit:** Fokus auf bewussten Konsum und Ressourcenschonung.

## Technologien
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Logik:** Intelligente Algorithmen zur Datumsberechnung
